# JarPacker
утилита для быстрой компоновки .jar файлов (fabric mod) в один .jar файл.

## Использование
```bash
.\JarPacker.exe jar=my_jar.jar input=input.jar --debug
```
(вы можете указывать любое кол-во jar=file, чтобы комповать сразу несколько файлов!)
на выходе получите input.jar-patch.jar
в котором будет находится my_jar.jar.
